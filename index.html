الشرح التفصيلي للكمبيوتر - Full PC Guide

الكمبيوتر يتكون من عدة مكونات رئيسية كل واحدة لها وظيفتها الخاصة.
A computer consists of several main components, each with its own function.

المعالج - CPU

المعالج هو العقل الرئيسي للكمبيوتر ويقوم بتنفيذ جميع العمليات الحسابية والمنطقية.
The CPU is the brain of the computer, executing all arithmetic and logical operations.

  • أنوية (Cores): تقوم بتنفيذ التعليمات - Execute instructions.
  • وحدة الحساب والمنطق (ALU): تنفذ العمليات - Performs calculations.
  • ذاكرة الكاش (Cache): لتسريع الوصول للبيانات - Speeds up data access.

كرت الشاشة - GPU

كرت الشاشة مسؤول عن الرسوميات والفيديوهات والألعاب.
GPU is responsible for graphics, videos, and games.

  • Shader Units: معالجة الرسوميات - Process graphics.
  • VRAM: تخزين مؤقت للصور والفيديوهات - Temporary storage of images/videos.

الذاكرة العشوائية - RAM

RAM تخزن البيانات التي يستخدمها الكمبيوتر بشكل مؤقت.
RAM stores data temporarily while computer is running.

  • Memory Chips: تخزن البيانات - Store data.
  • Memory Controller: يدير القراءة والكتابة - Manages reading/writing data.

اللوحة الأم - Motherboard

اللوحة الأم تربط جميع المكونات وتوزع الطاقة.
Motherboard connects all components and distributes power.

  • CPU Socket: مكان تثبيت المعالج - CPU slot.
  • RAM Slots: مكان تثبيت الذاكرة - RAM slots.
  • PCIe Slots: لتوصيل كروت الشاشة والأجهزة الأخرى - Expansion slots.
